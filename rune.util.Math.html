Class: Math

rune.util.Math()

The Math class contains a library of static methods for common mathematical calculations. The class can thus be used as an alternative to JavaScripts' built-in Math class.

Constructor

new Math()

Creates a new instance of the Math class. Note that this class is intended as a static library of methods, thus instantiation results in a warning.

Class: Tile

(package) rune.tilemap.Tile(width, height)

The rune.tilemap.Tile class represents a single Tile in a Tilemap. The class is internal and must therefore only be instantiated within its own package. The class is primarily used to calculate collisions between tiles and display objects.

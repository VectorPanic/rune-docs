Class: Config

rune.system.Config(optionsopt)

The Config class contains global settings for an application. The settings are used primarily during start-up to give the application the right conditions; resolution, image refresh rate, etc.

Constructor

new Config(optionsopt)

Creates a new configuration object. The options argument is used to override default settings.

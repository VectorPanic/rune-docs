Class: Canvas

rune.display.Canvas(widthopt, heightopt)

The Canvas class is a pixel buffer for 32-bit raster graphics and contains methods for manipulating the pixels included in the buffer. The class is used to represent all display objects and their current graphic state, ie. all display objects have their own pixel buffer (Canvas).

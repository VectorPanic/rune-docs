Class: Emitter

rune.particle.Emitter(xopt, yopt, widthopt, heightopt, optionsopt)

The Emitter class represents an emitter of particles; which can be used to create visual effects (such as smoke, explosions, etc.). An emitter is represented by a display object that can be added to the display list, each particle is its own display object, which is added to the same display list (parent object) as the emitter.

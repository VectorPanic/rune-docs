Class: Hitbox

rune.display.Hitbox(obj)

Represents a bounding box for an interactive object. Useful when you want to limit collision detection to a specific part of an interactive object. The class is primarily intended to be used internally by Rune. For example, the DisplayObject class uses an instance of Hitbox to do all of its collision handling.
